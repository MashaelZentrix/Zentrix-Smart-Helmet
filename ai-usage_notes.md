# AI Usage in ZENTRIX Development

Artificial intelligence was used as a development assistant during the design and development of the ZENTRIX smart safety helmet prototype.

AI supported the project in several areas:

• Structuring and organizing the Arduino code  
• Designing the system logic for multi-sensor integration  
• Suggesting threshold values for temperature, gas, and sound alerts  
• Improving readability and modular code structure  
• Assisting in debugging and troubleshooting during development  
• Helping document the project clearly for demonstration and evaluation

The student remained responsible for the main engineering decisions including:

• Selecting hardware components  
• Connecting and testing sensors  
• Adjusting the code according to real hardware behavior  
• Designing the overall safety concept and prototype logic

AI acted as a technical assistant to accelerate development while the final implementation and validation were performed through real hardware testing.
